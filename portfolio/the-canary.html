Embedded System built for the safety of miners

Project information

  • Category: Embedded Systems
  • Client: Embedded Systems, Ed Stott, Imperial
  • Project date: March, 2022
  • Project URL: https://github.com/simon-staal/TheCanary
  • Marketing Website: https://kc31949.wixsite.com/the-canary
  • Technologies: Python, MERN, MQTT, AWS

This was a project for a 3rd Year module created in teams of 4. The task was to create a novel embedded system using a number of sensors provided to us and a raspberry pi. Our idea was a device attachable to a mining helmet which sends data to a remote control point which is capable of monitoring multiple devices and displaying data that is measured. The webapp was created using the MERN stack and python was used on the pi.
