Issie Demonstations

Project information

  • Category: Software Development
  • Client: High Level Programming, Thomas Clarke, Imperial
  • Project date: March, 2022
  • Project URL: https://tomcl.github.io/issie/
  • Technologies: F# - Functional Programming, Visual Studio

As part of our High Level Programming Module in 3rd Year we were tasked with developing parts of ISSIE an open source educational software for digital circuit design and simulation developed by students and maintained by Tom Clarke. Built using F# and the .NET framework, the project saw me developing functional programming skills as well as gaining experience working on an existing codebase. My part involved allowing symbols to align with each other and allowing for the rotation and flipping of symbols. Every group in the year were given the same tasks but as a culmination of the effort of my 4-strong group our changes were chosen to be integrated with the latest version.
